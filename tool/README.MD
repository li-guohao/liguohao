# 介绍

小豪的工具箱站点

# 技术栈

* Java
* freemarker
* SpringBoot
* SpringDataJPA
* MySql
* Hibernate

# QQ机器人功能列表
* /命令：查询所有命令
* /AV号或BV号：查询视频信息
* /番剧番剧名：查询番剧信息
* /服务器状态IP(域名):端口：查询Minecraft服务器信息
